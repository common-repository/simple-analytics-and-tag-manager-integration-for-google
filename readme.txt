The Simple Google Analytics and Google Tag Manager Integration adds the tracking code automatically to your website's source code. Current data privacy conditions (DSGVO) is considered if you want to activate.

== Description ==

The Simple Google Analytics and Google Tag Manager Integration plugin adds tracking code based on your options.

Based on research we adjusted tracking code to fit with current DSGVO. But of course we do not take any responsibility to correctness. To ensure the correctness please consult your lawer. 


== Installation ==
= From your WordPress dashboard =

1. Visit 'Plugins > Add New'.
2. Search for 'Simple Google Analytics and Google Tag Manager Integration'.
3. Activate Simple Google Analytics and Google Tag Manager Integration from your Plugins page.

= From WordPress.org =

1. Download Simple Google Analytics and Google Tag Manager Integration plugin.
2. Upload the 'simple-google-analytics-and-tag-manager-integration' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Simple Google Analytics and Google Tag Manager Integration from your Plugins page.

= Once Activated =

1. Visit WordPress dashboard sidebar in 'Settings > Google Analytics / Google Tag Manager'.
2. Check visible Options and enter Google Analytics and/pr Google Tag Manager Tracking Code.
3. Once you have clicked the Saved Button, tracking code will be added to your website's source code.

== Frequently Asked Questions ==

= GTM is activated but <noscript> is not visible below <body> tag. =

You should have at least Wordpress 5.2 and your activated theme must support the hook 'wp_body_open'.
